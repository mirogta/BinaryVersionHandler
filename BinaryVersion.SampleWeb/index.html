BinaryVersion.Handler Sample Website

C# HttpHandler which shows versions of binaries used by currently running ASP.NET application

Add this to your web.config to enable the BinaryVersion.Handler. 

    <system.webserver>
        <handlers>
            <remove name="version_xml" />
            <remove name="version_json" />
            <add name="version_xml" verb="*" path="version.xml" type="BinaryVersion.Handler.VersionHandler, BinaryVersion.Handler" />
            <add name="version_json" verb="*" path="version.json" type="BinaryVersion.Handler.VersionHandler, BinaryVersion.Handler" />
        </handlers>
    </system.webserver>

See http://www.iis.net/configreference/system.webserver/handlers for more information about handlers.
